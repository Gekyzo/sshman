Installation

Prerequisites

  • Java 21 or higher
  • Maven 3.6 or higher
  • ssh-keygen (usually pre-installed on Linux/macOS)

Build from Source

# Clone the repository
git clone https://github.com/Gekyzo/sshman.git
cd sshman

# Build the project
./build.sh

# Run sshman
./sshman --help

Add to PATH (Optional)

# Add this to your ~/.bashrc or ~/.zshrc
export PATH="$PATH:/path/to/sshman"

# Then you can run from anywhere
sshman --help
